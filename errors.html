CVM Error Values

The following errors are defined:

  • CVME_GENERAL (value 1): Unspecified error.
  • CVME_BAD_CLIDATA (value 2): Invalid data from the client.
  • CVME_BAD_MODDATA (value 3): Invalid data from the CVM.
  • CVME_IO (value 4): Input or output error while communicating with the CVM.
  • CVME_PERMFAIL (value 100): The credentials were accepted by the module but are incorrect.
Any value other than CVME_PERMFAIL (100) should be considered a temporary error.
