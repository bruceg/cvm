CVM SQL Queries

Substitution: All occurrences of the percent symbol (%) are replaced with the quoted value of the account name. Do not include quotes in the query.

Results: The query must produce the following fields exactly in order:

  1. Password, encrypted with crypt.
  2. Actual account name. Must not be null.
  3. User ID (integer). Must not be null.
  4. Group ID (integer). Must not be null.
  5. Real name. Optional.
  6. Directory. Must not be null.
  7. Login shell. Optional.

Default: The default query is "SELECT password, username, userid, groupid, realname, directory, shell FROM accounts WHERE username=%".
